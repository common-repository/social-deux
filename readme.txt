== Description ==

Social Deux is a plugin that gives you 11 popular custom styled social media sharing buttons with counters. Here you can set basic setting like facebook app id, VKontakte App ID. It comes with lot of styling feature like you can resize the size of button. The plugin give you option to place the social button to 11 places with custom styling feature.

1) Style setting tab give you change the color and radius of the button for all places.
2) Mail setting tab give you option to set mail id with many option as well.
3) Placement setting tab give you option to select the the buttons for each placements with many option like button width, size and font.
4) Fan count setting tab give you more option.
5) Under Advanced setting tab you can set the height and width of the button popup.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `social-deux` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Navigate to setting Social Deux under setting menu.

That's it! Now you can easily start to place button to differrent places in WordPress
